# ZOHO-CRM-PROJECT
This project is a simple ASP.NET Core Web API that integrates with Zoho CRM. It accepts lead information through a REST API and automatically creates a new Lead record in Zoho CRM using the Zoho CRM REST API.

The project demonstrates how to authenticate with Zoho CRM, send HTTP requests, and deploy the application using Docker.

Technologies Used
ASP.NET Core Web API (.NET)
C#
Zoho CRM REST API
HttpClient
JSON
Docker
Postman
